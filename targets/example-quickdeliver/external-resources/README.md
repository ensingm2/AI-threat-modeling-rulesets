# QuickDeliver - On-Demand Delivery Platform

## Overview

QuickDeliver connects customers, delivery drivers, and merchants for fast on-demand delivery. Think Uber Eats or DoorDash but simpler.

## What It Does

- Customers order food/items from local merchants via mobile app
- Drivers pick up and deliver orders
- Merchants receive and fulfill orders
- Everyone gets paid
