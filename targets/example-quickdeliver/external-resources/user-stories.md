# QuickDeliver - User Stories

## Customer Stories

**Register & Login**
- Sign up with email or phone
- Login with Google/Apple/Facebook
- Save payment methods (Stripe handles it)
- Save delivery addresses

**Browse & Order**
- Browse restaurants by cuisine, rating, distance
- Search for specific food or restaurant
- Add items to cart with modifiers (size, toppings, etc.)
- Apply promo codes
- Choose ASAP or schedule delivery
- Add tip
- Place order

**Track Order**
- See order status (Confirmed → Preparing → Picked Up → Delivered)
- Watch driver on map in real-time
- Get push notifications for status changes
- Call/text driver if needed

**After Delivery**
- Rate food and driver
- Report problems (missing items, wrong order, late)
- Request refund
- Reorder previous orders

**Subscription**
- Subscribe to QuickDeliver Plus ($10/month)
- Get free delivery on all orders
- Cancel anytime

---

## Driver Stories

**Onboarding**
- Apply with driver's license and vehicle info
- Upload insurance and registration
- Pass background check
- Set up bank account for payments

**Work**
- Go online/offline whenever
- Get delivery requests with earnings estimate
- Accept or decline (15 seconds to decide)
- Navigate to restaurant
- Pick up order
- Navigate to customer
- Take photo of delivery
- Mark delivered

**Earnings**
- See today's earnings in real-time
- Cash out instantly ($0.50 fee) or wait for weekly payout
- View earnings history

**Stats**
- See rating, acceptance rate, total deliveries
- Read customer feedback

---

## Merchant Stories

**Setup**
- Apply with business license
- Set up profile (logo, hours, delivery area)
- Connect bank account for payouts

**Menu**
- Add menu items with photos and prices
- Create categories and modifiers
- Mark items out of stock
- Import menu from CSV or POS system (Square, Toast)

**Orders**
- Get notified of new orders
- Accept or reject
- Mark when order is ready
- See driver arrival time

**Money & Stats**
- See sales by day/week/month
- Track top-selling items
- View payout history (weekly deposits)
