# QuickDeliver - Mobile App Features

## Customer App (iOS + Android)

### Account
- Sign up with email or social (Google/Apple/Facebook)
- Save addresses and payment methods
- Profile with name and photo

### Browse & Order
- Search restaurants by food type or name
- Filter by rating, delivery time, price
- View menus with photos
- Add items to cart with customizations (size, toppings, etc.)
- Apply promo codes
- Choose delivery time (now or later)
- Add tip

### Track Order
- Real-time map showing driver location
- Order status updates (Confirmed → Preparing → Picked Up → Delivered)
- ETA countdown
- Push notifications for each stage
- Call/text driver

### After Delivery
- Rate food and driver
- Report problems and get refunds
- Reorder previous orders
- View order history

### QuickDeliver Plus
- $10/month for unlimited free delivery
- Manage subscription

## Driver App (iOS + Android)

### Setup
- Apply with driver's license and vehicle info
- Upload insurance docs
- Bank account for payouts

### Driving
- Go online/offline toggle
- Get delivery requests (15 sec to accept)
- See earnings estimate before accepting
- Navigate to restaurant (Google Maps)
- Mark order picked up
- Navigate to customer
- Take photo of delivery
- Mark delivered

### Earnings
- See real-time earnings today
- Daily/weekly/monthly totals
- Cash out instantly ($0.50 fee) or wait for weekly deposit
- View performance stats (rating, acceptance rate)

## Common Features

- Push notifications
- GPS tracking
- In-app calling/texting
- Photo upload
- Works offline for basic stuff (view history)
